Generate session

Authenticated Request Flow

1

Browser sends request with cookie

GET /dashboard
Cookie: _your_app_session=eyJzZXNzaW9u...
2

Rails extracts cookie

ActionDispatch::Cookies reads cookie
↓
Encrypted value: 'eyJzZXNzaW9u...'
3

Decrypt session ID

MessageEncryptor.decrypt_and_verify(cookie)
↓
session_id = '8f3a2c1b9d4e5f6a7b8c9d0e1f2a3b4c'
4

Fetch from Memcache

cache_key = 'sessions:8f3a2c1b...'
raw_data = Memcache.get(cache_key)
↓
Found: true
5

Deserialize session data

session_data = Marshal.load(raw_data)
↓
{
  session_id: '8f3a...',
  warden.user.user.key: [[123], '$2a$...']
}
6

Warden authenticates user

user_id = session_data['warden.user.user.key'][0][0]
↓
user_id = 123
7

Load user from database

User.find(123)
↓
Return user object

Set current_user SUCCESS 

@current_user = user
↓
Request authenticated ✓
Controller action executes

Logout Flow

1

User clicks logout

DELETE /users/sign_out
Cookie: _your_app_session=eyJzZXNz...
2

Rails processes logout

Devise::SessionsController#destroy
↓
Extract session_id from cookie
3

Delete from Memcache

cache_key = 'sessions:8f3a2c1b...'
Memcache.delete(cache_key)
↓
Session data removed ✓
4

Clear cookie in browser

Set-Cookie: _your_app_session=; expires=Thu, 01 Jan 1970
↓
Browser deletes cookie

User logged out SUCCESS 

current_user = nil
✓ No cookie
✓ No session in Memcache
✓ User must login again
